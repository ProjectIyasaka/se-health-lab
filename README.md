# se-health-lab

ITエンジニア向け健康アフィリエイトブログの運用用リポジトリです。

## ドキュメント

- [コマンド一覧](./docs/command-list.md)

## よく使うコマンド

```powershell
cd C:\Users\haruc\projects\se-health-lab
npm run dev
npm run tweets
npm run tweet-dashboard
```

## 補足

- X運用の管理データは `content/social/x-posts.json`
- 豆知識・質問は `tweet-dashboard` から半自動で投稿
